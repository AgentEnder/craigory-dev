PR Digest

Generate a comprehensive digest of a GitHub pull request, optimized for AI agent handoffs.

Prefer the command line?

Install pr-digest and run it directly from your terminal:

npx pr-digest --url https://github.com/owner/repo/pull/123

Or auto-detect from a local repo checkout:

cd your-repo && npx pr-digest

Run npx pr-digest --help for all options including --output, --token, and --ai-provider.
